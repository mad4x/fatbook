E' necessario possedere docker e docker compose installati sul pc.

Una volta copiata la repo eseguire il comando:


docker compose up --build -d




--- PORTE SU CUI GIRANO I SERVIZI ---


localhost:3000 --> frontend (Next.js)


localhost:8080 --> backend (Spring Boot)


localhost:5050 --> pgadmin


localhost:5432 --> database postgres
